# Braun-Multiplier-NxN
This project is a Verilog simulation of an NxN Braun Multiplier.
The Braun multiplier is a parallel array multiplier used specifically for unsigned numbers. Its primary appeal lies in its regular, modular structure, which makes it incredibly easy to layout on a silicon chip (VLSI) and scale for different bit-widths.
